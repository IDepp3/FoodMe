Instructivo de FoodMe

Inicio

En la siguiente imagen vemos el menu principal de la aplicacion, en el cual nos da la posibilidad de registrarnos o loguearnos en el sistema.

Login

Al presionar el numero 1 se nos mostrara el Login, el cual nos permitira acceder a la aplicacion mediante un Usuario y Contraseña.

Registro

En caso de no existir el cliente, lo que debemos hacer es presionar el numero 2 (Registro) y rellenar los siguientes campos para registrarnos en el sistema.

Cliente

Una vez logueado en el sistema, tendremos a nuestra disposicion el menu del cliente el cual cuenta con las siguientes opciones.

Hacer reserva

Al momento de crear una reserva el sistema nos dara una serie de campos que debemos completar para que la reserva sea creada con exito.
El sistema nos pedira informacion sobre la fecha del evento y la hora del mismo, tambien nos pedira que le indiquemos el menu deseado y la cantidad de personas de dicho evento, por ultimo pedira una breve descripcion sobre datos que querramos hacerle saber al sistema.

Ver perfil

La opción "Ver Perfil" nos mostrara nuestra información personal con todos los datos correspondientes.

Editar perfil

La opcion "Editar Perfil" nos brinda distintas opciones para poder modificar un dato en concreto, sin la necesidad de tener que cambiar todos los datos.

Listar reserva

Al seleccionar la opcion "Historial de Reservas" el sistema nos dara la posibilidad de poder listar y ver las reservas realizadas.

Menu Administrador

En la siguiente imagen vemos el menu del administrador de nuestra aplicacion. El cual cumple una funcion importante, ya que cuenta con funciones muy importantes para el sistema.
Como por ejemplo : Gestionar empleados, Gestionar clientes, Gestionar reservas y por ultimo, Gestion menu.

Gestion Empleados

En el menu de Gestion de Empleados podremos : Dar de alta un empleado, Editar un empleado y por ultimo tenemos la posibilidad de Inhabilitar dicho empleado.

Al seleccionar "Dar de alta Empleado" el sistema nos dara la opcion de crear un nuevo empleado en el sistema, como asi tambien nos dejara dar de alta a un empleado que se encuentra en el sistema pero estaba inhabilitado.

Crear Empleado

Al momento de crear un nuevo empleado, el sistema nos pedira que completemos los datos personales del empleado a crear.
Si los datos ingresados son correctos, el sistema nos indicara mediante un cartel, que el empleado fue creado con exito

Buscar Empleado

El administrador tambien puede buscar un determinado empleado, el sistema le solicitara que ingrese el dni del empleado a buscar. Si el dni corresponde a un empleado del sistema, se mostrara dicho empleado con sus datos personales.

En caso de que el dni no corresponda a ningun empleado, el sistema arrojara un mensaje notificando que no hay ningun empleado con ese dni.

Inhabilitar Empleado

En el caso de la funcion "Dar de baja un Empleado" el administrador ingresara el dni del Empleado que quiere dar de baja, si el dni que ingreso corresponde a un empleado del sistema, su estado pasara de ACTIVO a INACTIVO.

Listar Empleados

Si el administrador desea ver el listado de los empleados que hay en el sistema, lo puede hacer mediante la opcion "Listar Empleados" dicha opcion mostrara todos los empleados con sus respectivos datos personales.

Eliminar Reserva

El administrador tambien tiene la posibilidad de eliminar una reserva del sistema.
El sistema le pedira al administrador que ingrese el ID de la reserva que desea eliminar.
En caso que el ID coincida con la reserva cargada en el sistema, dicha reserva se eliminara y nunca se llevara a cabo el evento.

Listar Reservas

Si el administrador quisiera saber cuales son las reservas que se encuentran en el sistema, puede hacerlo a traves de la opcion "Listar Reserva".
Dicha opcion mostrara todas las reservas que se encuentran en el sistema con sus respectivos datos.
